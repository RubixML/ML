# Softsign
A function that squashes the input smoothly between -1 and 1.

### Parameters
This activation function does not have any parameters.

### Example
```php
use Rubix\ML\NeuralNet\ActivationFunctions\Softsign;

$activationFunction = new Softsign();
```
