# Lottery
Hold a lottery in which each category has an equal chance of being picked.

**Data Type:** Categorical

### Parameters
This strategy does not have any parameters.

### Example
```php
use Rubix\ML\Other\Strategies\Lottery;

$strategy = new Lottery();
```
