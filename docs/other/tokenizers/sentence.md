# Word Tokenizer
This tokenizer matches sentences starting with a letter and ending with a punctuation mark.

## Parameters
This tokenizer does not have any parameters.

## Example
```php
use Rubix\ML\Extractors\Tokenizers\Sentence;

$tokenizer = new Sentence();
```
